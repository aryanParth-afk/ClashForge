# UI/UX Design Brief
## Project Name: ClashForge

### 1. Design Philosophy
We are avoiding generic "AI-generated" tropes (no glowing neon borders on pitch-black backgrounds, no floating 3D tech shapes). We are using a **Premium SaaS / FinTech aesthetic applied to gaming**. It must feel like a professional data analytics tool.

### 2. Design System Guidelines
* **Backgrounds:** Monochromatic dark slate/charcoal backgrounds (`#0B0F19`, `#1E293B`).
* **Accent Colors:** High-contrast tactical colors.
  - Primary Accent: Industrial amber/gold (`#F59E0B`) for resources and primary actions.
  - Secondary/Destructive: Muted crimson (`#991B1B`) for war stats or alerts.
  - Success/Complete: Muted sage green (`#15803D`) for maxed buildings.
* **Typography:** Strict sans-serif layout. 
  - UI Text: **Inter** or **Plus Jakarta Sans** (clean, modern).
  - Numbers & Data: **JetBrains Mono** or similar monospaced font for displaying resource numbers, tags, and timers perfectly aligned.

### 3. Data Presentation
* Do not use overly massive circular progress bars. Use linear, compact progress strips.
* Use complex data tables with inline sorting, filtering, and custom row badges.
* Incorporate micro-interactions: slight hover states on table rows, smooth transitions on tabs, and loading skeletons instead of basic spinner wheels.
